一、14个品质维度 — 谁负责、是否必须

#维度谁做必须?说明
1视觉质感
渐变/阴影/光泽		 Claude (CSS) 必须 CSS radial-gradient 可做金属感。专业级需要 Figma 设计稿导出 SVG
2图标素材
奖品/按钮/装饰		 Kenney + 你提供 必须 Kenney Board Game Icons (250个CC0), Board Game Pack (卡片/骰子)
特殊图标需要你提供或 AI 生成
3字体排版 Claude (CSS) 必须 Google Fonts Outfit + 字重系统。已做完
4动效动画 Claude (CSS + JS) 高优先 CSS @keyframes 做基础动画。Lottie 可做复杂动画但需要 AE 导出的 JSON
5粒子特效 canvas-confetti 必须 已集成 CDN。永远不需要自己做粒子
6音效设计 Kenney Casino Audio + ZzFX 高优先 ZzFX 做 UI 音效(点击/成功)
Kenney Casino Audio (54个CC0) 做真实音效(转动/硬币/骰子)
7触觉反馈 navigator.vibrate 可选 免费 API，已集成。非必须但加分
8间距节奏 设计系统 必须 design-system.css 已定义令牌。Claude 执行
9色彩体系 设计系统 必须 已做完。品牌紫+暖金强调
10状态覆盖 Claude (JS) 必须 loading/empty/error/success 四个状态
11文案质量 Claude 高优先 Claude 直接生成。不需要外部工具
12背景氛围 Claude (CSS) + 你提供素材 可选 CSS 渐变/光晕能做基础。纹理图需要设计稿或素材
13连贯过渡 Claude (CSS) 高优先 CSS transition + ease 曲线。不需要外部工具
14无障碍 Claude 必须 reduced-motion / 对比度 / 触控目标

结论: 14个维度中，11个 Claude 直接能做，1个需要素材(Kenney免费)，2个需要你提供

需要你做的: 图标素材(如果Kenney的250个不够用)、背景纹理(如果CSS不够)。

其他全部: Claude + 开源组件 + 免费素材。零付费、零API Key、零外部模型。

四、总结: 需要你做的只有两件事

你需要做的:

1. 下载 Kenney 素材 — 去 kenney.nl 下载 Board Game Pack + Board Game Icons + Casino Audio + Interface Sounds。解压放到项目 assets/ 目录。

2. 确定图标准则 — 告诉我是用 Kenney 的 250 个图标，还是你需要更特定风格的。

我做的:

素材到位后，我负责: 替换所有 emoji 图标为 SVG、集成真实音效替换 ZzFX 合成音、用卡片/骰子素材美化游戏组件、升级 builder-v2 编辑器。
