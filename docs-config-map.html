配置化分两层：①「左侧面板配置」——用户在游戏页内手动调参，刷新后重置；②「URL配置（?c=）」——配置编码进链接，发给任何人打开即生效，是对外开放的关键。
绿色边框= 已支持URL配置。金色标签= 核心可配项。后续优先把高价值游戏的左侧面板配置项搬进URL参数。
